App: demo-app

Kubernetes Cluster Orchestrator: AWS EKS/FARGATE

IaaC: terraform

Package Manager: helm

Kubernetes CLI: kubectl

Container Driver: Docker

This is sample container app (utilizes FastAPI framework), let's call it "demo-app", it fetches secrets from secret manager and availabe at /secrets and fetches ssm from Parameter Store and available at /ssm

The test paths are:


    Values from secret mananger
    Values from parameter store
    Get the hostname
    Application Liveness HealthCheck
    Application Readiness HealthCheck
    Deployed Image Id

A Helm Chart to release it to Kubernetes Cluster for demo/learning purposes.
